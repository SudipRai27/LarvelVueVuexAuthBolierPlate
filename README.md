# About

A Laravel Vue Authentication boiler plate using JWT

#### `Backend: Laravel 8`

#### `PHP: 7.4`

#### `MYSQL: 5.7`

#### `Vue: 2.6`

# Installation

-   Clone the repo
-   Run composer install
-   Cp .env.example to .env
-   Generate application key (php artisan key:generate)
-   Run npm install
-   Run npm run dev
-   Configure .env file
-   Run migrations (php artisan migrate)

# Generate JWT Secret

> php artisan jwt:secret

# Running the project

> php artisan serve
